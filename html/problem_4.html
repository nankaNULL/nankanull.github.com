Description

企鹅老师出了一道简单的数学题：“输入 n 个正整数，输出其中偶数的和”。你能用程序编写出来吗？

Input

第一个整数是n(1 < n < 100),代表有 n 个正整数，接下来输入 n 个正整数(每个数都小于1000)。

Output

输出一个整数，表示 n个正整数中偶数的和。

Samples

input:
9 1 2 3 4 5 6 7 8 9
output:
20
